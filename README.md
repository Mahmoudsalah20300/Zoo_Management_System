# 🐾 Zoo Management System (zOOP)

## Overview
This project is a command-line-based zoo management system implemented in C++.  
It simulates managing a zoo on a grid-based map where users can build enclosures, buy animals, and interact with the system using commands.

---

## Features
- Grid-based map with ASCII visualization  
- Build enclosures (Savanna, Forest, Aquarium)  
- Buy and manage animals (Lion, Giraffe, Gorilla, Dolphin)  
- Shop system using a configuration file  
- Zoo statistics and rating system  
- Interactive command-line interface  
- Error handling for invalid inputs  
---

## OOP Concepts Used
- Classes and objects  
- Inheritance (Animal base class and derived classes)  
- Polymorphism (virtual functions)  
- Encapsulation (private/protected members)  
- Constructors and destructors  
- STL (`std::vector`, `std::string`, file streams)
